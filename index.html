To get started, you'll need a Flickr API key and OAuth tokens. Click below to enter your credentials.

Flickr Credentials

Register a free app at flickr.com/services/apps/create to get your API key and secret.
Get your OAuth token using the Flickr API Explorer:
1. Visit the link above while logged in to Flickr
2. Click "Sign In" then "Call Method"
3. Copy the oauth_token from the response URL shown in your browser's address bar after authorization, or use the token tool below.
Find yours at flickr.com/services/api/explore/flickr.test.login — it appears in the response as user_nsid.
